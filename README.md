## 📊 Overview
Power BI dashboard analyzing road accidents
across India from 2018 to 2022.

## 🛠️ Tools Used
- Power BI
- Microsoft Bing Maps

## 📌 Key Insights
- 3000 total road accidents recorded
- 7366 total fatalities
- Clear weather has highest accidents 21%
- Auto-Rickshaw involved in most accidents
- Accidents increased from 2018 to 2022
